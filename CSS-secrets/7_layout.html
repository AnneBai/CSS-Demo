垂直居中的方法

绝对定位-margin:auto, 适用于宽高固定

请把我居中
.center1 { position: relative; } .center1 .child { position: absolute; margin: auto; top:0; bottom:0; left:0; right:0; }

绝对定位-translate, 适用宽高不固定

请把我居中
.center1-1 { position: relative; } .center1-1 .child { position: absolute; top:50%; left: 50%; transform: translate(-50%, -50%); }

flex

请把我居中
.center2 { display: flex; justify-content: center; align-items: center; }

translate-适用已知父元素高度, 相对视窗垂直居中时可使用vh

请把我居中
.center3 .child { margin-top: 90px; margin-left: 50%; transform: translate(-50%, -50%); }
